STEP 2 - Finding information about the creators of the artwork: Giambologna and Tommaso Laureti

QUERY 5

With the following query we wanted to ask the question “What are the cultural events provided by Giambologna in Bologna?” In order to have a complete picture of how active the artist was in Bologna. 
PREFIX cis: <http://dati.beniculturali.it/cis/>
PREFIX clv: <https://w3id.org/italia/onto/CLV/>
PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>

SELECT DISTINCT ?culturalevent ?eventlabel ?author ?authorlabel ?place
WHERE {
  ?culturalevent a cis:CulturalEvent ;
                rdfs:label ?eventlabel .
  ?culturalevent cis:involvesCulturalEntity ?culturalEntity .
  ?culturalEntity a-cd:hasAuthor ?author .
  ?author rdfs:label ?authorlabel
  FILTER(REGEX(?authorlabel, "Giambologna", "i"))
  ?place a cis:GeographicalFeature ;
         rdfs:label "Bologna"
}

Results:

We get a long list of interesting results. The following “CulturalEvent” is the most useful for us as it lists different artworks by Giambologna:
Giambologna, gli dei, gli eroi. Genesi e fortuna di uno stile europeo nella scultura

This is an interesting result because although it shows Giambologna’s works in Firenze, the descriptions help us understand that the artist was closely linked to the de' Medici family and realized many sculptures on their commission.



Here are some works by Giambologna, all to be found in different collections in Firenze:


Among the results, we find another IRI of the artist with his French name Jean de Boulogne as he was also known:
IRI Jean de Boulogne detto Giambologna

We also find a (possible) portray of the artist:
Ritratto del Giambologna (?) (scultura - busto) by Tacca Pietro (attribuito) (sec. XVII):
IRI Ritratto del Giambologna



This result also provides historical information that tells us that there used to be a similar portrait/sculpture of the artist in the Pinacoteca di Bologna but unfortunately it went missing.

All the other results obtained by running query 5 confirm the above analysis, that is to say, that Giambologna was a very influential artist of his time, who worked for ‘de Medici family and was very active in Firenze (XVI-XVII century).

QUERY 6

Still, we want to give it another try and use the keyword UNION to ask for information about cultural properties associated with Giambologna in either Bologna or Florence. We add the keyword LIMIT to focus on a specific amount of results.
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>

SELECT DISTINCT ?cp
WHERE {
  {
    ?cp a arco:HistoricOrArtisticProperty ;
        rdfs:label ?l .
    FILTER(REGEX(?l, "Giambologna", "i"))
    FILTER(REGEX(?l, "Bologna", "i"))
  }
  UNION
  {
    ?cp a arco:HistoricOrArtisticProperty ;
        rdfs:label ?l .
    FILTER(REGEX(?l, "Giambologna", "i"))
    FILTER(REGEX(?l, "Firenze", "i"))
  }
}
ORDER BY ASC(?cp)
LIMIT 100

Results:



The query works: we get a long list of results.
Some of the results show us that his works are present in other northern Italian cities too, especially in Genova. Here are some of his works kept in collections in other Italian cities:



Some very interesting results we hadn’t found before regarding our topic are the following:
Statua di Nettuno di Giambologna sulla confezione di Idrolitina con zampilli d'acqua (locandina) by Majani Augusto detto Nasica (primo quarto sec. XX)

IRI Statua di Nettuno - confezione Idrolitina



IRI Statua di Nettuno - confezione Idrolitina (n.2)



These images were used to advertise the product Idrolitina, produced in Bologna by the company Gazzoni&C. As we learned from an article by the Corriere della Sera. The advertising images were created by the Bolognese artist Majani Augusto also known as Nasica.

IRI Majani Augusto

These results confirm that Giambologna and Laureti’s work Fontana del Nettuno was also a symbol of the city in the advertising industry in the first half of the XX century.

QUERY 8

To be more exact we run the COUNT query: this SPARQL query is asking for the total number of cultural events in Florence that involve Giambologna as the author. 
PREFIX cis: <http://dati.beniculturali.it/cis/>
PREFIX clv: <https://w3id.org/italia/onto/CLV/>
PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
SELECT COUNT (?culturalevent ) AS ?n
WHERE
{
  ?culturalevent a cis:CulturalEvent .
  ?culturalevent cis:involvesCulturalEntity ?culturalEntity .
  ?culturalEntity a-cd:hasAuthor ?author .
  ?author rdfs:label ?authorlabel.
  ?place a cis:GeographicalFeature;
          rdfs:label "Firenze".
  ?culturalevent clv:hasSpatialCoverage ?place.

  FILTER( REGEX(?authorlabel, "Giambologna" , "i"))
}

Results:


Instead, for Bologna the results are 0.
Therefore, it is evident that the ArCo ontology lacks information about other works by Giambologna in Bologna.

QUERY 9

Before moving onto the second author of the Fontana del Nettuno, we want to run the query CONSTRUCT. This SPARQL CONSTRUCT query is useful for creating new RDF triples that represent information about cultural events in Florence involving Giambologna as the author.
We tried the same query asking about cultural events in Bologna but it was empty. 
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX clv: <https://w3id.org/italia/onto/CLV/>
PREFIX cis: <http://dati.beniculturali.it/cis/>
PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>
PREFIX l0: <https://w3id.org/italia/onto/l0/>

CONSTRUCT {
  ?place a cis:GeographicalFeature ; l0:name "Firenze" .
  ?event a cis:CulturalEvent ; clv:hasSpatialCoverage ?place ; l0:name ?eventName ;
         cis:involvesCulturalEntity ?culturalProperty .
  ?culturalProperty a-cd:hasAuthor ?author ; l0:name ?title .
  ?author l0:name ?authorName
}
WHERE {
  ?place a cis:GeographicalFeature ; rdfs:label "Firenze" .
  ?event a cis:CulturalEvent ; clv:hasSpatialCoverage ?place ; rdfs:label ?eventName ;
         cis:involvesCulturalEntity ?culturalProperty .
  ?culturalProperty a-cd:hasAuthor ?author ; rdfs:label ?title .
  ?author rdfs:label ?authorName
  FILTER(REGEX(?authorName, "Giambologna", "i"))
}

QUERY 10

So now that we have gathered all the information there was to gather about Giambolgna (at least within the ArCo context) we want to know whether there is a resource classified as a person with a label containing Tommaso Laureti, the other author of the fountain, within the ontology.
For this purpose, we use the QUERY ASK. 
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX cpv: <https://w3id.org/italia/onto/CPV/>

ASK
WHERE {
  ?subject a cpv:Person;
           rdfs:label ?label

  FILTER(REGEX(?label, "Tommaso Laureti", "i"))
}

Results:

The answer is “false”. Whereas if we try with Giambologna it gives us “true” as an answer.
Therefore, there is another important piece of information missing in the ontology and we want to fill this gap with the help of the LLMs.
