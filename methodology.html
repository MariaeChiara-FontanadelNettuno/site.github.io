METHODOLOGY

  • Choosing a topic of interest
  • Using the ArCo SPARQL endpoint to find information about the artwork in the ArCo ontologies
  • Building queries containing all SPARQL keywords in order to find a variety of details about the artwork
  • Analyzing the queries’ results to gather knowledge about our topic from our research in ArCo
  • Identifying gaps of knowledge in the ArCo documentation
  • Engineering prompts for Gemini and ChatGPT by using 4 prompting techniques to learn more about the artwork
  • Comparing and analyzing the LLMs’ outputs
  • Transforming the new information into RDF format with the LLMs’ help
  • Designing triples based on ArCo and on the new information
  • Using HTML and CSS to create the website with the help of the software Bluefish link to the free download
  • Publishing the website on GitHub
