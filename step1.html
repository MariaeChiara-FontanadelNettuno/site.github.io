STEP 1 - Finding the Fontana del Nettuno in ArCo and gathering information about it

QUERY 1

First of all, we used this query to check whether there is an entity in ArCo representing this artwork. 
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>

SELECT DISTINCT ?cp
WHERE {
?cp a arco:HistoricOrArtisticProperty ;
rdfs:label ?l .
FILTER(REGEX(?l, "Fontana del Nettuno", "i"))

}
  • The keyword DISTINCT serves to eliminate duplicates.
  • FILTER and REGEX are useful keywords to get more specific information about our topic: the FILTER keyword in SPARQL is used to filter the results based on certain conditions. It allows you to specify a condition that must be met for a result to be included. The REGEX function is used to perform regular expression matching on the specified string.

    QUERY 2


    Then, we used the following query to find more material about the Fontana del Nettuno. In fact, this query is searching for unique resources that are classified as "Subjects" (according to the ArCo ontology) and have a name or title that includes the phrase "Fontana del Nettuno". 
    PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT ?subject ?label
WHERE {
  ?subject a a-cd:Subject ;
           rdfs:label ?label .
 
  FILTER (REGEX(?label, "Fontana del Nettuno", "i"))
}

    QUERY 3


    We then ran the queries below in order to find properties and values associated with our chosen artwork in the ArCo ontologies. In addition, identifying properties and values can help us at a later stage to design the triples we need.

    The Fontana del Nettuno is present in other cities in Italy too, so we added the FILTER ‚ÄúBologna‚Äù and noticed that by doing so the query gave us more accurate results on our topic: 
    PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>
PREFIX cis: <http://dati.beniculturali.it/cis/>

SELECT DISTINCT ?property ?value
WHERE {

  ?cp a arco:HistoricOrArtisticProperty ;
      rdfs:label ?l ;
      ?property ?value .
  FILTER(REGEX(?l, "Fontana del Nettuno", "i"))
  FILTER(REGEX(?l, "Bologna", "i"))
}

    QUERY 4


    We ran the query with the keyword OPTIONAL in order to find further depictions and representations of our artwork. In the following query we also removed DISTINCT and let the
    * instead in order to get all possible results. 
    PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>

SELECT *
WHERE {

  ?cp a arco:HistoricOrArtisticProperty ;
      rdfs:label ?l .
  FILTER(REGEX(?l, "Fontana del Nettuno", "i"))
  FILTER(REGEX(?l, "Bologna", "i"))
  OPTIONAL { ?culturalProperty a-cd:depiction ?depiction }
}

    Results:

    The same happens if we change the query and ask with the OPTIONAL block about the material, technique, or legal situation of the artwork.
